# Ultimate Weapon Icons

Four icons for the ultimate weapons, in the brand "ultimate purple" (`#b48cff`).

## Files
- `*.svg` — 24×24 viewBox, vector, uses `currentColor` so you can tint via CSS `color:` (web)
  or by find/replacing `currentColor` with a hex in an engine that doesn't support it.
- `png/*.png` — 256×256 transparent PNGs, pre-colored #b48cff, for engines that prefer raster.

## Icons
| File | Weapon |
|---|---|
| freeze-bomb | Freeze Bomb (snowflake) |
| emp-burst | EMP Burst (radial lightning burst) |
| time-warp | Time Warp (clock) |
| laser-beam | Laser Beam (flare beam) |

Stroke icons (freeze, time, laser) scale cleanly at any size; re-render PNGs larger if needed.
